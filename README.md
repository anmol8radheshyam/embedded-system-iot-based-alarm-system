# Motion Detection Alarm System using PIR Sensor

## Overview

This project is developed using Arduino UNO and PIR Motion Sensor. The system detects human motion and activates an LED and buzzer to provide an alert.

## Components Used

* Arduino UNO
* PIR Motion Sensor
* LED
* Buzzer
* Breadboard
* Jumper Wires

## Working Principle

1. PIR sensor continuously monitors movement.
2. When motion is detected, a signal is sent to Arduino UNO.
3. Arduino activates the LED and buzzer.
4. The system returns to monitoring mode when no motion is detected.

## Features

* Real-time motion detection
* Audio and visual alerts
* Low-cost security solution
* Easy implementation

## Applications

* Home Security

* Smart Surveillance Systems
